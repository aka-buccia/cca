# CCA - Choreography Correctness Analyzer
Choreography Correctness Analyzer is a static correctness analyzer for FaaSChalCore choreographies. FaaSChal is a coreographic programming language tailored for  serverless Function-as-a-Service architectures. FaaSChalCore is a choreographic calculus containing the key features of FaaSChal, and for which a static analysis discipline is defined.
